webMCR Add-on SkinPoser v1.0

Установка:

1. Скопировать файлы из папки 'upload' в папку с webMCR
1.1 [для UNIX систем] Выдать всем файлам и каталогам webMCR из подпапки instruments/sp2/ права на чтение \ запись
2. Перейти по ссылке your_site.com/?mode=skinposer , где 'your_site' - адрес вашего сайта
